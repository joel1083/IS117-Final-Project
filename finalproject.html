What is Git/Github/Markdown?

Git: a version-control system for tracking changes in source code during the stages of software/web development.

GitHub: a web hosting service that operates under the version control system called Git.

Markdown: is a lightweight markup language with plain text formatting syntax. Its design allows it to be converted to many output formats, but the original tool by the same name only supports HTML. Markdown is often used to format readme files, for writing messages in online discussion forums, and to create rich text using a plain text editor.

Accessing NJIT's AFS

OpenAFS Client: The OpenAFS client can be used to natively access files in AFS. By using the OpenAFS Client, you avoid the need to log into another host to access those files.

WebStorm: An IDE that can utilize OpenAFS and can access and manage files through SSH.

HTML/CSS/BOOTSTRAP

HTML: (Hypertext Markup Language) a standardized system of using various series of tags to create pages on the world wide web.

CSS: (Cascading Style Sheet) CSS are used to format the layout of Web pages. They can be used to define text styles, table sizes, and other aspects of Web pages that previously could only be defined in a page's HTML.

BootStrap: Created by Twitter to simplify the way websites are created.
